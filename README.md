# Saturnx Website

A modern, responsive website for Saturnx - a Saudi-based technology & engineering company.

## Features

- **Fully Responsive Design**: Optimized for all devices from 320px to 4K displays
- **Mobile-First Navigation**: Hamburger menu with smooth slide-in animation for mobile devices
- **Touch-Optimized**: All interactive elements are sized for easy touch interaction (minimum 44px)
- **Performance Optimized**: Reduced animations on mobile for better performance
- **Modern UI**: Dark premium design with gradient effects and smooth transitions

## Mobile Responsiveness

The website is fully optimized for mobile phones with the following breakpoints:

### Breakpoints
- **Desktop**: 1024px and above
- **Tablet**: 900px - 1023px
- **Mobile**: 520px - 899px
- **Small Mobile**: 375px - 519px
- **Extra Small**: 320px - 374px

### Mobile Features
1. **Hamburger Menu**: 
   - Appears on screens < 900px
   - Smooth slide-in animation from right
   - Full-screen overlay
   - Close on ESC key or clicking outside
   - Prevents body scroll when open

2. **Touch-Friendly**:
   - All buttons minimum 44px height
   - Larger tap targets
   - Visual feedback on touch

3. **Optimized Content**:
   - Scalable typography
   - Flexible images
   - No horizontal overflow
   - Word wrapping and hyphenation

4. **Performance**:
   - Reduced animation duration on mobile
   - Optimized transitions
   - Hardware acceleration where appropriate

## Testing on Mobile

1. **Using Browser DevTools**:
   - Open DevTools (F12)
   - Toggle device toolbar (Ctrl+Shift+M)
   - Test various device sizes

2. **On Real Devices**:
   - Serve the website locally or on a server
   - Open on mobile devices
   - Test all interactions and navigation
